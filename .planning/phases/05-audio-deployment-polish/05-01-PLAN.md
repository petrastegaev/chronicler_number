  <action>
    Create `frontend/src/audio/SoundManager.ts` following the singleton pattern from `backend/connection_manager.py` (module-level instantiation, exported singleton).

    The class must implement:

    1. **`SoundName` type** -- string literal union of `'tick' | 'tick_fast' | 'end_round' | 'winner'`

    2. **`private sounds: Map<SoundName, Howl>`** -- stores preloaded Howler.js instances

    3. **`private preloaded = false`** -- guard against double-preload

    4. **`preload(): void`** method:
       - Guard: return early if `this.preloaded` is true
       - Define config array with all 4 entries: name + src path `/sounds/{name}.mp3`
       - For each config: `new Howl({ src: [src], preload: true, volume: 0.5 for tick/tick_fast, 1.0 for end_round/winner, onloaderror callback logging `[Audio] Failed to load ${name}: code ${errCode}` })`
       - Set `this.preloaded = true`

    5. **`play(name: SoundName): void`** method:
       - Look up sound in map; if not found, `console.warn('[Audio] Sound "${name}" not found')` and return
       - Call `sound.play()`
       - Howler.js autoUnlock handles AudioContext resume; if AudioContext is still suspended, Howler queues the play

    6. **`stopAll(): void`** method:
       - Iterate map values, call `.stop()` on each

    7. **`stop(name: SoundName): void`** method:
       - Look up sound, call `.stop()` if found

    8. **Export singleton** at module level: `export const soundManager = new SoundManager()`

    Import path: `import { Howl } from 'howler'` -- howler is already in package.json and available in node_modules.

    Conventions to follow (from PATTERNS.md):
    - Singleton instantiated at module level like ConnectionManager
    - TypeScript typed throughout (no `any` for SoundName or Howl)
    - Error handling with console.warn (like connection_manager.py try/except pattern)
  </action>

    - `import { useEffect, useRef } from 'react'`
    - `import { useGameStore } from '../stores/gameStore'`
    - `import { soundManager } from './SoundManager'`

    The hook uses `useRef` for tracking previous values and an initialization guard:
    - `const initializedRef = useRef(false)`

    Four `useEffect` subscriptions (following Zustand subscribe() API from RESEARCH.md and PATTERNS.md):

    1. **Preload on mount**: Single useEffect with empty deps, calls `soundManager.preload()`, sets `initializedRef.current = true`.

    2. **Timer tick subscription** -- Subscribe to `state.remaining`:
       - Use `useGameStore.subscribe((state) => state.remaining, (remaining, prevRemaining) => {...})`
       - Guard: only play on decrement: `if (remaining < prevRemaining)`
       - If `remaining > 3`: `soundManager.play('tick')`
       - Else: `soundManager.play('tick_fast')`
       - Cleanup: return `unsub`

    3. **Round result subscription** -- Subscribe to `state.roundResult`:
       - Use `useGameStore.subscribe((state) => state.roundResult, (roundResult) => {...})`
       - If `roundResult !== null`: `soundManager.play('end_round')`
       - Cleanup: return `unsub`

    4. **Game end subscription** -- Subscribe to `state.gameEndResult`:
       - Use `useGameStore.subscribe((state) => state.gameEndResult, (gameEndResult) => {...})`
       - If `gameEndResult !== null`: `soundManager.play('winner')`
       - Cleanup: return `unsub`

    5. **Phase reset subscription** -- Subscribe to `state.phase`:
       - Use `useGameStore.subscribe((state) => state.phase, (phase) => {...})`
       - If phase is `'idle'` or `'waiting'`: `soundManager.stopAll()`
       - Cleanup: return `unsub`

    Export: `export function useSoundEffects() { ... }`
