# Quick Task: Figma Design + CoFo Fonts + Rename

Применить дизайн-систему из Figma-макета: заменить цветовую палитру, шрифты (Inter → CoFo Sans), переименовать игру.

## Дизайн-система (из PDF/Figma)

### Цвета
| Токен | Старый | Новый | Hex |
|-------|--------|-------|-----|
| `--color-wb-bg` | `#1a0a2e` | `#161616` | Почти чёрный фон |
| `--color-wb-surface` | `#2d1b4e` | `#2a2a2a` | Тёмно-серая поверхность |
| `--color-wb-text` | `#eeeeee` | `#d5ccde` | Светло-фиолетовый текст |
| `--color-wb-text-muted` | `#9ca3af` | `#8a7a9a` | Приглушённый фиолетовый |
| `--color-player1` | `#3B82F6` | `#7f30e3` | Фиолетовый (Player 1) |
| `--color-player2` | `#EF4444` | `#ff00fe` | Неоново-розовый (Player 2) |
| `--color-correct` | `#10B981` | `#40f99b` | Неоново-зелёный |
| `--color-warning` | `#F59E0B` | `#f59e0b` | Без изменений |
| `--color-danger` | `#EF4444` | `#ff3333` | Красный |

### Шрифты
- Заголовки/основной: **CoFo Sans Pixel** (пиксельный — игровая тематика)
- Текст/body: **CoFo Sans Regular** (читаемый для параграфов)

### Название
- "Дуэль чисел" → "Число летописца"

## Tasks

### Task 1: Шрифты — установка и подключение
- **Files:** `frontend/src/index.css`, `frontend/src/assets/fonts/`
- **Action:** 
  1. Переместить CoFoSansPixel.otf и CoFoSansRegular.otf в `frontend/public/fonts/`
  2. Заменить `@font-face` блоки в `index.css`: Inter → CoFo Sans
  3. Обновить `--font-sans`: 'CoFo Sans Pixel', 'CoFo Sans Regular', system-ui, sans-serif
  4. TimerRing.tsx: обновить fontFamily с Inter → CoFo Sans Pixel
- **Verify:** Собрать фронтенд без ошибок, проверить что шрифты загружаются

### Task 2: Цвета — новая палитра
- **Files:** `frontend/src/index.css`
- **Action:**
  1. Заменить все `--color-*` токены в `@theme` на новые значения
  2. Добавить `--color-player1-light` и `--color-player2-light` для hover-состояний
- **Verify:** Визуально проверить контраст, собрать без ошибок

### Task 3: Переименование игры
- **Files:** `frontend/src/components/JoinScreen.tsx`, `frontend/src/components/GameHeader.tsx`, `frontend/src/components/admin/GameControlTab.tsx`
- **Action:** Заменить все вхождения "Дуэль чисел" на "Число летописца"
- **Verify:** `grep -r "Дуэль чисел" frontend/src/` не находит совпадений
