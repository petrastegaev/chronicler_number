    - `setPhase`, `setPlayer1Nickname`, `setPlayer2Nickname`, `setPlayer1Online`, `setPlayer2Online`

    - `setGameCount`, `setQuestions`, `setTotalQuestions`, `setCsvResult`, `setToken`

    - `setScoreUpdate(p1Score, p2Score)` — sets both scores
    - `resetForRestart()` — sets phase 'lobby', scores to 0, currentRound to 0, but keeps nicknames (per Pitfall 1 in RESEARCH.md)
    - `reset()` — returns to initial state (used for disconnect)

    Create `frontend/src/hooks/useAdminWebSocket.ts` per D-05 (separate hook, NOT reusing useWebSocket):
    - Import `useCallback`, `useEffect`, `useRef` from react
    - Import `useAdminStore` from stores/adminStore
    - Import type `WsMessage` from types/ws

    Function `useAdminWebSocket()`:
    - `wsRef = useRef<WebSocket | null>(null)`
    - `intentionalCloseRef = useRef(false)`
    - `connect()` callback:
      - Create WebSocket to `ws://${host}/ws` (or wss for https)
      - `ws.onopen`: store ws ref in adminStore, send join message: `{event: "join", data: {role: "admin"}}`
      - `ws.onmessage`: parse JSON, dispatch by event name. Handlers:
        - `joined`: set phase 'lobby', set token from msg.data.token
        - `game_started`: set phase 'playing', set player1Nickname, player2Nickname from data
        - `round_started`: set currentRound, totalRounds from data
        - `timer_tick`: (admin may receive these — ignore or store for display)
        - `score_update`: set player1Score, player2Score from data
        - `game_end`: set phase 'finished'
        - `game_reset`: call `resetForRestart()` — keep nicknames, clear scores+round
        - `player_joined`: read player2_nickname from data, set player2Nickname
        - `error`: log via console.warn
      - `ws.onclose`: if not intentionalCloseRef.current, set phase 'connecting', ws null
    - `startGame()` callback: sends `{event: "start_game", data: {}}` over ws
    - `restart()` callback: sends `{event: "restart", data: {}}` over ws
    - `useEffect` cleanup: set intentionalCloseRef to true, close ws

    Return `{ connect, startGame, restart }` from the hook.

    Follow the exact same pattern as `useWebSocket.ts` — wsRef, intentionalCloseRef, protocol detection, cleanup pattern.
